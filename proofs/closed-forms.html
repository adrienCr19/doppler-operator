N8 — Generating closed forms of higher-index Doppler operators from the \(\Do_{000}\) base case

The rapidity ODE (N3) generates Taylor series to any order. For closed forms valid at all \(\beta\) one instead climbs up from the \(\Do_{000}\) base case using three verified identities: the seed behind \(\Do_{000}\), the \(\ell\)-raising recursion (B10), and transposition + parity (B3). The result is a complete algorithm and a structure theorem: every \(m=0\) Doppler-operator element is a rational operator function built from one universal special function. All formulas on this page are verified in verification/06_closed_forms.wl (checks F1–F8, PASS at \(10^{-25}\)–\(10^{-28}\) including \(\beta = 0.7\), i.e. genuinely non-perturbative).

0  The universal function family

Definitions
\[ J_w \;\equiv\; \frac{(\gamma+p)^w - (\gamma-p)^w}{w} \;=\; \frac{2\sinh(w\eta)}{w}, \qquad E_w \;\equiv\; (\gamma+p)^w + (\gamma-p)^w \;=\; 2\cosh(w\eta), \]

using \(\gamma \pm p = e^{\pm\eta}\) (rapidity \(\eta\)). Key properties: \(J_{-w} = J_w\), \(E_{-w} = E_w\), \(J_0 = 2\eta\), \(E_0 = 2\), \(E_1 = 2\gamma\), \((\gamma+p)(\gamma-p) = 1\), and the product-collapse rule (check F6, exact):

\[ J_{a+q}\,J_{b+q} \;=\; \frac{E_{a+b+2q}\; -\; E_{a-b}}{(a+q)(b+q)}, \]

whose second term is \(q\)-independent — this is the mechanism by which products of two kernel factors collapse into single-difference closed forms.

1  The base case, rewritten

The seed of everything is the kernel element behind \(\Do_{000}\) (Seed 1 of N7):

\[ S(d) \equiv {}^{d}\Kk^0_{00}(-\beta) = \frac{(\gamma+p)^{1-d} - (\gamma-p)^{1-d}}{2(1-d)p} = \frac{J_{1-d}}{2p}, \]

so that on power laws (\(\Oh \to -q\))

\[ \Do_{000}(q) = \frac{1}{\gamma}\,S(-1-q)\,S(-q) = \frac{J_{2+q}\,J_{1+q}}{4\gamma p^2} \;\overset{\text{collapse}}{=}\; \frac{E_{3+2q} - E_1}{4(2+q)(1+q)\,\gamma p^2} = \frac{(\gamma+p)^{3+2q} + (\gamma-p)^{3+2q} - 2\gamma}{4(2+q)(1+q)\,\gamma p^2}, \]

recovering [CR26, Eq. 10a] (checks F5, C11). The \(J\)-product form is the one that generalizes.

2  Step one: raise the kernel indices (B10)

The \(\ell\)-raising recursion (B10) expresses \({}^{d}\Kk^0_{\ell 0}\) through \({}^{d}\Kk^0_{\ell-1,0}\), \({}^{d-1}\Kk^0_{\ell-1,0}\), \({}^{d}\Kk^0_{\ell-2,0}\): each step adds one power of \(1/p\) and shifts the weight window by one. Starting from \(S(d) = J_{1-d}/2p\), every kernel column element is a finite \(J\)-combination with polynomial coefficients. One step (\(C_1 = 1/\sqrt3\)):

\[ {}^{d}\Kk^0_{10}(-\beta) = -\sqrt3\,\frac{\gamma S(d) - S(d-1)}{p} = -\frac{\sqrt3}{2p^2}\Big(\gamma J_{1-d} - J_{2-d}\Big) , \]

(check F1). A second step (\(C_2 = 2/\sqrt{15}\), \(C_1/C_2 = \sqrt5/2\), using \(\gamma^2 = 1+p^2\)):

\[ {}^{d}\Kk^0_{20}(-\beta) = \frac{\sqrt5}{4p^3}\Big[(3+2p^2)\,J_{1-d} - 6\gamma\,J_{2-d} + 3\,J_{3-d}\Big], \]

(check F2) — equivalent to [CR26, Eq. 9b]. Iterating gives the general pattern

\[ {}^{d}\Kk^0_{\ell 0}(-\beta) = \frac{1}{2\,p^{\ell+1}} \sum_{t=0}^{\ell} a_{\ell t}(\gamma, p^2)\; J_{1-d+t}, \]

with polynomial coefficients \(a_{\ell t}\) produced mechanically by B10. Elements \({}^{d}\Kk^0_{\ell\ell'}\) with \(\ell' > 0\) follow by raising the other index through transposition (B3: \({}^{d}\Kk^0_{\ell\ell'}(-\beta) = {}^{2-d}\Kk^0_{\ell'\ell}(+\beta)\)) plus parity — exactly the moves used in N5/N6.

3  Step two: assemble and collapse

On power laws every Doppler-operator element is a product of two such finite \(J\)-combinations:

Assembly
\[ \Do^0_{\ell\ell'\ell''}(q) = \frac{1}{\gamma}\; \underbrace{{}^{-1-q}\Kk^0_{\ell\ell'}(+\beta)}_{\text{via B3/parity/B10: } \sum_t \alpha_t\, J_{2+q+t}\,/\,p^{\#}} \;\; \underbrace{{}^{-q}\Kk^0_{\ell'\ell''}(-\beta)}_{\text{via B10: } \sum_t \beta_t\, J_{1+q+t}\,/\,p^{\#}} . \]

4  Worked closed forms (new)

\(\Do_{101}\)verified F3 (incl. β = 0.7)
\[ \Do^0_{101}(\nu,\beta) \;=\; \frac{3}{4\gamma p^4}\, \Big(\gamma J_{2-\Oh} - J_{3-\Oh}\Big)\Big(\gamma J_{1-\Oh} - J_{\Oh}\Big). \]
\(\Do_{020}\)verified F4 (incl. β = 0.7)
\[ \Do^0_{020}(\nu,\beta) \;=\; \frac{5}{16\gamma p^6} \Big[(3+2p^2) J_{2-\Oh} - 6\gamma J_{1-\Oh} + 3 J_{\Oh}\Big] \Big[(3+2p^2) J_{1-\Oh} - 6\gamma J_{2-\Oh} + 3 J_{3-\Oh}\Big]. \]

Here \(J_{w-\Oh}\) is the operator function \(\big[(\gamma+p)^{w-\Oh} - (\gamma-p)^{w-\Oh}\big]/(w-\Oh)\); on \(\nu^q\), \(\Oh\to-q\) recovers the scalar formulas (note \(J_{\Oh} = J_{-\Oh}\) by evenness). Expanding either product with the collapse rule turns it into a sum of single terms \(\frac{E_{n+2q} - \text{const}}{(a+q)(b+q)}\), i.e. into rational operator functions of \(\Oh\) acting through simple frequency rescalings \((\gamma\pm p)^{-\Oh} n(\nu) = n((\gamma\pm p)\nu)\) — the same structure [CR26, Eq. 10a] has, now for every element. Together with thermal averaging over \(p\) this gives the exact (all-order in temperature) building blocks of the SZ signal.

5  Consistency checks built into the forms

  • Reflection (N1). Under \(q \to -3-q\), evenness gives \(J_{2+q} \to J_{-1-q} = J_{1+q}\) and \(J_{q} \to J_{-3-q} = J_{3+q}\): in both worked forms the two brackets simply swap, so the diagonal elements are manifestly invariant (check F7, exact zero).
  • Parity (N2). \(J_w\) and \(E_w\) are even in \(\beta \to -\beta\) up to the explicit \(p\)-powers; counting powers of \(p\) reproduces \((-1)^{\ell+\ell''}\).
  • Series limit. Expanding \(\Do_{020}\) in \(p\) reproduces the ODE-generated table entry \(\tfrac{1}{45}\Oh(\Oh^3-6\Oh^2+5\Oh+12)\,p^4 + \mathcal O(p^6)\) (check F8).
  • Poles are removable. Apparent poles at \(w=0\) (i.e. \(\Oh\) hitting an integer) are removable: \(J_0 = 2\eta\); the closed forms are entire in \(\Oh\).

6  Structure theorem

Every element is hyperbolic-rational

For \(m = 0\) and any \((\ell,\ell',\ell'')\), the Doppler operator has the exact form

\[ \Do^0_{\ell\ell'\ell''}(\nu,\beta) \;=\; \frac{1}{\gamma\, p^{\,\ell+\ell''+2}} \sum_{\text{finite}} c_{ab}(\gamma, p^2)\; \frac{E_{a+b-2\Oh} - E_{a-b}}{(a-\Oh)(b-\Oh)}, \qquad E_{w} = 2\cosh(w\eta), \]

with integer index pairs \((a,b)\) in the window \(a \in \{2,\dots,\ell+2\}\), \(b \in \{1,\dots,\ell''+1\}\) shifted by the \(\ell'\)-raising, and polynomial coefficients \(c_{ab}\) generated by B10. In rapidity variables everything is elementary: \(\cosh\)/\(\sinh\) of \((\text{integer} - 2\Oh)\eta\) over polynomial denominators. This is the closed-form counterpart of the statement that the ODE (N3) generates the same elements perturbatively.
