N13 — The \(\ell'\) generating function: general phase functions and the Henyey–Greenstein round trip

The middle index \(\ell'\) of \(\Do_{\ell\ell'\ell''}\) is the multipole of the rest-frame scattering phase function: Thomson scattering contributes the fixed weights \(\{w_0, w_2\} = \{1, \tfrac1{10}\}\). This page promotes \(\ell'\) from an index to a variable: the generating function over \(\ell'\) is itself a physical round-trip operator, and every axisymmetric phase function is handled by one identity.

Statement new verified 10⁻¹¹ (3D quadrature)

Let \(p(\cos\Theta)\) be any axisymmetric rest-frame scattering kernel, and let the round trip with phase function \(p\) be \(\mathcal O_p = \frac1\gamma\,{}^{-1}\Bo(\nu,-\beta)\circ \mathsf P\circ{}^{0}\Bo(\nu,\beta)\), where \(\mathsf P n(\hat y) = \int p(\hat y\cdot\hat y'')\,n(\hat y'')\,\dd\hat y''\). Then its multipole matrix elements are

\[ \big[\mathcal O_p\big]^m_{\ell\ell''} \;=\; \sum_{\ell'} w_{\ell'}\, \Do^m_{\ell\ell'\ell''}(\nu,\beta), \qquad w_{\ell'} = 2\pi\!\int_{-1}^{1} p(x)\,P_{\ell'}(x)\,\dd x \quad\text{(Funk–Hecke eigenvalues)}. \]

In particular, for the Henyey–Greenstein kernel with asymmetry parameter \(t\),

\[ p_t(x) = \frac{1}{4\pi}\,\frac{1-t^2}{\big(1 + t^2 - 2tx\big)^{3/2}} \qquad\Longrightarrow\qquad w_{\ell'} = t^{\ell'} \text{ exactly}, \]

so that the \(\ell'\)-generating function of the Doppler operators is the HG round trip:

\[ \sum_{\ell'} t^{\ell'}\, \Do^m_{\ell\ell'\ell''}(\nu,\beta) \;=\; \big[\mathcal O_{p_t}\big]^m_{\ell\ell''}, \]

with the explicit position-space representation (on \(\nu^q\); \(D_+(\mu) = \gamma(1+\beta\mu)\)):

\[ \big[\mathcal O_{p_t}\big]^m_{\ell\ell''}(q) = \frac1\gamma \iint \dd\hat y_1\,\dd\hat y_2\; \bar Y^*_{\ell m}(\hat y_+[\hat y_1])\,\bar Y_{\ell'' m}(\hat y_+[\hat y_2])\; p_t(\hat y_1\!\cdot\hat y_2)\; D_+(\mu_1)^{-3-q}\, D_+(\mu_2)^{\,q}. \]

Proof

(i) Funk–Hecke. Any axisymmetric kernel acts diagonally on spherical harmonics: \(\int p(\hat y\cdot\hat y'')Y_{\ell'm'}(\hat y'')\dd\hat y'' = w_{\ell'}Y_{\ell'm'}(\hat y)\) with \(w_{\ell'}\) the Legendre coefficient above. Hence \(\mathsf P = \sum_{\ell'm'} w_{\ell'}|\ell'm'\rangle\langle\ell'm'|\), and sandwiching between the two boosts replaces the fixed projector of the Doppler-operator definition by the weighted sum — the first display. The HG eigenvalues are the classical generating-function property \(\sum_\ell \tfrac{2\ell+1}{4\pi}t^\ell P_\ell(x) = p_t(x)\). check G0 (exact)

(ii) Position space. Insert the verified position-space representations of the two kernel factors (as in the N1 proof): \({}^{a}\Kk_{\ell\ell'}(+\beta) = \int\dd\hat y_1\,\bar Y^*_{\ell m}(\hat y_+[\hat y_1])\bar Y_{\ell'm}(\hat y_1)D_+(\mu_1)^{a-2}\) and \({}^{b}\Kk_{\ell'\ell''}(-\beta) = \int\dd\hat y_2\,\bar Y^*_{\ell'm}(\hat y_2)\bar Y_{\ell''m}(\hat y_+[\hat y_2])D_+(\mu_2)^{-b}\), with \((a,b) = (-1-q,-q)\); summing \(\sum_{\ell'm'}w_{\ell'}Y_{\ell'm'}(\hat y_1)Y^*_{\ell'm'}(\hat y_2)\) reconstitutes the kernel \(p(\hat y_1\cdot\hat y_2)\) by the addition theorem. \(\blacksquare\)

Direct numerical check

At \((\ell,\ell'') = (0,0)\), \(m = 0\), \(t = 0.4\), \(q = \tfrac12\), \(\beta = 0.3\): the truncated multipole sum \(\sum_{\ell' \le 16} t^{\ell'}\Do_{0\ell'0}\) (kernel quadratures; tail \(< 10^{-32}\)) equals the direct 3D quadrature of the HG round trip to \(1.2\times10^{-11}\) relative (quadrature-limited). check G1

Consequences

  • Any phase function at once. Klein–Nishina angular corrections, Rayleigh scattering, dust — any axisymmetric kernel enters only through its Funk–Hecke weights \(w_{\ell'}\); the frequency physics is carried entirely by the universal \(\Do_{\ell\ell'\ell''}\) elements already generated by N3/N5/N6/N10. Thomson is the special case \(w = \{1, 0, \tfrac1{10}, 0, \dots\}\) (check G2), reproducing \(\hat{\mathcal S}_{\rm th} = \Do_{000} + \tfrac1{10}\Do_{020} - 1\).
  • The sum rule is the \(t\to1\) limit. As \(t\to1\), \(p_t \to \delta^{(2)}\) on the sphere and the round trip degenerates to the pure boost-in/boost-out combination: the generating function reproduces N4's \(\sum_{\ell'}\Do = \delta_{\ell\ell''} - \beta[\dots]\).
  • Single-quadrature channels. For forward-peaked media the \(t\)-resummed operator is available directly from the 2D/3D integral above without ever enumerating \(\ell'\) — useful when many multipoles contribute (e.g. resonant-line or dust-scattering transfer with Doppler boosts).
  • Analytic structure in \(t\). \(\sum t^{\ell'}\Do_{0\ell'0}(q)\) is analytic in \(t\) inside the unit disk with Taylor coefficients given by our closed forms — providing generating-function routes to summed identities (e.g. weighted moments \(\sum \ell'(\ell'+1)t^{\ell'}\Do\) via \(t\partial_t\)).
