N22/N23 — Closed forms for every spin and every \(m\): the rationalization lemma

The site's closed-form calculus (N8: two-factor \(J\)-family closed forms) was built at \(s = m = 0\). This page removes both restrictions at once: every kernel element \({}_s\Kk^{d,m}_{\ell\ell'}\) — any spin weight, any azimuthal number — is a finite \(J\)-combination, by one structural fact. harness 30_spin_m_closed_forms.wl (Z1–Z5)

1  The rationalization lemma

Lemma (the kernel integrand is always rational) new

Write the spin-weighted harmonic in Wigner-\(d\) form, \({}_sY_{\ell m} \propto (\sin\tfrac\theta2)^{|m+s|}(\cos\tfrac\theta2)^{|m-s|}\,R_{\ell}(\mu)\) with \(R_\ell\) a polynomial. In the kernel both harmonics carry the same \((s,m)\), and on the aberration curve the half-angle factors pair up rationally:

\[ \sin\tfrac{\theta_+}{2} = \sin\tfrac\theta2\,\sqrt{\tfrac{1-\beta}{1+\beta\mu}},\qquad \cos\tfrac{\theta_+}{2} = \cos\tfrac\theta2\,\sqrt{\tfrac{1+\beta}{1+\beta\mu}}, \] \[ \Longrightarrow\quad {}_sY_{\ell m}(\mu)\,{}_sY_{\ell' m}(\mu_+) \;=\; (1{-}\beta)^{\frac{|m+s|}2}(1{+}\beta)^{\frac{|m-s|}2}\; \frac{\text{polynomial}(\mu)}{(1+\beta\mu)^{\max(|m|,|s|)}} . \]

So the integrand of \({}_s\Kk^{d,m}\) is polynomial\((\mu)\times(1+\beta\mu)^{-d-k}\) with integer \(k\) — exactly the \(s=m=0\) structure. With \(t = 1+\beta\mu\) and \(\gamma(1\pm\beta) = e^{\pm\eta}\), \(\int t^{\,n-d}\dd t = \gamma^{\,d-n-1} J_{n+1-d}\), and the reduction is a two-line algorithm:

\[ {}_s\Kk^{d,m}_{\ell\ell'}(\beta) \;=\; \frac{2\pi}{\beta}\,(1{-}\beta)^{\frac{|m+s|}2}(1{+}\beta)^{\frac{|m-s|}2} \sum_n e_n\,\gamma^{-n-1}\,J_{n+1-d}, \qquad J_w = \frac{2\sinh(w\eta)}{w}, \]

with rational coefficients \(e_n\) read off from the polynomial. Implemented and verified against direct quadrature at \(10^{-19}\): the spin-2 columns \(\ell = 2\ldots4\), \(m = 0,1,2\) (Z1) and the general-\(m\) scalar columns (Z2).

2  N22: the polarized closed-form catalog

The polarized SZ system needs the \(\ell' = 2\) columns at \(s = 2\) and their round trips. All are now exact; e.g. (\(d\) symbolic):

\[ {}_2\Kk^{d,1}_{22}(-\beta) = \frac{5(\coth\eta - 1)\,\operatorname{csch}\eta}{2(d{+}1)d(d{-}1)(d{-}2)(d{-}3)} \Big[3e^{-d\eta}(1{+}\coth\eta)^2(d{-}1{+}2\coth\eta)\;+\;e^{d\eta}\big(3{+}d{+}2(d{-}3)d^2 - 6(d{-}2)d\coth\eta + 9(d{-}1)\coth^2\eta - 6\coth^3\eta\big)\Big], \]

with the remaining columns (\(\ell = 2\ldots4\), \(m = 0,1,2\)) produced by the same two-line reduction (harness, Z1). Every polarized Doppler element then follows in closed form, \(T(s_1,s_2)^m_{\ell 2\ell''}(q) = \tfrac1\gamma\,{}_{s_1}\Kk^{-1-q,m}_{\ell 2}(+\beta)\, {}_{s_2}\Kk^{-q,m}_{2\ell''}(-\beta)\) — verified against quadrature for \(T(2,2), T(0,2), T(2,0)\) at \(m = 0,1,2\) (Z3), including the \(m\)-summed mixed diagonals of [RC26, Eq. 20] (Z4). The operator-level derivation of these same forms is now supplied by the polarized ladders N24 (spin-weighted N5/N6, climbing from the \(T^m_{222}\) seeds); the \(J\)-reduction of this page serves as the independent cross-check of that climb (Y3). The polarized sector now has the same status as the scalar one: seeds → recursions → closed forms → exact thermal averages, with nothing left as a series.

3  N23: general-\(m\) closed forms — the \(J\)-calculus is universal

The open structural question (does the \(J\)-family survive at \(m \ge 1\), or do the closed forms become genuinely hypergeometric?) is settled by the same lemma: at \(s = 0\) the prefactor is \((1-\beta^2)^{|m|/2} = \gamma^{-|m|}\) — rational — so the general-\(m\) elements are \(J\)-family closed forms with an extra \(\gamma^{-|m|}\), no new transcendents. For example:

\[ {}_0\Kk^{d,1}_{11}(-\beta) \;=\; \frac{3}{2p^3\,(d-1)(d-2)}\Big[\gamma p\,E_d \;-\; \big(1 + d\,p^2\big)\,J_d\Big], \qquad E_d = 2\cosh d\eta,\quad J_d = \frac{2\sinh d\eta}{d}, \]

(so the \(m=1\) elements mix \(J\) and \(E\) at weight \(d\) itself — one \(\gamma^{-|m|}\)-type factor and no new functions).

Two structural remarks. (i) This is the closed-form face of the exchange symmetry (N19): \({}_0\Kk^{d,m} = {}_m\Kk^{d,0}\), so a general-\(m\) scalar element is a spin-\(m\) element at \(m=0\), and both sit in the same \(J\)-class. (ii) The \(m\)-seeds' \({}_2F_1\) forms (B11/N7) are therefore \(J\)-reducible at integer arguments — the hypergeometric structure is only apparent at generic complex indices.
