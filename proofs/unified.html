The unified index-raising formulation

Every index-raising result on this site — the outer recursions (N5/N6), the middle-index relation (N10), the closed forms (N8), the kinematic element (N16) — and every result of the independent generation paper (its four ladders, gauge identity, pure-physical relations P0–P4, factorization, \(F\)-brackets) is one statement seen from different sides. This page is that statement: one object, one seed, one recurrence (in two mirror slots), three symmetries. All boxed equations are verified end-to-end in verification/21_unified.wl (\(s=0\), \(m=0\)).

The one object

The bracket \(\Ff\) — a single boost-operator column
\[ \Ff_{ab}(q) \;\equiv\; {}^{-1-q}\Kk_{ab}(+\beta) \qquad(a = \text{row},\; b = \text{column};\ \text{a weight-}(-1-q)\text{ aberration kernel}). \]

The entire Doppler operator is a bilinear form in this one family V1, exact:

\[ \boxed{\;\Do_{\ell\ell'\ell''}(q) \;=\; \frac{(-1)^{\ell'+\ell''}}{\gamma}\; \Ff_{\ell\ell'}(q)\;\Ff_{\ell'\ell''}(q-1)\;} \]

(the two scattering legs are \(\Ff_{\ell\ell'}(q)\) and, via parity, \(\Ff_{\ell'\ell''}(q-1)\)). Raising any of the three physical indices \(\ell, \ell', \ell''\) is therefore raising an index of \(\Ff\).

The one seed

\(\Ff_{00}\), with \(J_w = \dfrac{(\gamma+p)^w-(\gamma-p)^w}{w} = \dfrac{2\sinh w\eta}{w}\)
\[ \Ff_{00}(q) \;=\; \frac{J_{2+q}}{2p} \qquad\Big(\text{general } m:\ \Ff \text{ seeded by } {}^{d}\Kk^m_{mm} = \gamma^{-(m+d)}\,{}_2F_1\big(\tfrac{m+d}{2},\tfrac{m+d+1}{2};\tfrac32{+}m;\beta^2\big)\Big). \]

The one recurrence (two mirror slots)

A single three-term recurrence raises an index of \(\Ff\); applied to the row or the column it gives the two forms below, which are transposes of each other under the \(\Ff\)-reflection. Here \(C_\ell = \ell/\sqrt{4\ell^2-1}\) (\(m=0\)); each rule references only smaller indices, and its lone companion is the same bracket at a neighbouring \(q\):

Raise the row \(a\) V2a
\[ \Ff_{ab}(q) = \frac{1}{C_a\,p}\Big(\gamma\,\Ff_{a-1,b}(q) - \Ff_{a-1,b}(q{+}1)\Big) - \frac{C_{a-1}}{C_a}\,\Ff_{a-2,b}(q). \]
Raise the column \(b\) V2b
\[ \Ff_{ab}(q) = -\frac{1}{C_b\,p}\Big(\gamma\,\Ff_{a,b-1}(q) - \Ff_{a,b-1}(q{-}1)\Big) - \frac{C_{b-1}}{C_b}\,\Ff_{a,b-2}(q). \]

The companion \(q\)-shift (\(+1\) for the row, \(-1\) for the column) is the weight-lattice shift of the generation paper (its \((j,k)\) label), collapsed to a \(q\)-shift by the gauge identity \({}^{(j+1,k+1)}X(q) = {}^{(j,k)}X(q+1)\) — so the extended \((j,k)\) lattice is not a separate structure, just these \(q\)-shifted companions.

The three symmetries

Reflection, parity, evenness V3
\[ \textbf{(reflection)}\quad \Ff_{ab}(q) = (-1)^{a+b}\,\Ff_{ba}(-4-q), \qquad \textbf{(evenness)}\quad J_{-w} = J_w, \] \[ \textbf{(Doppler reflection, N1)}\quad \Do_{\ell\ell'\ell''}(q) = \Do_{\ell''\ell'\ell}(-3-q), \qquad \textbf{(parity, N2)}\quad \Do(-\beta) = (-1)^{\ell+\ell''}\Do(\beta). \]

The \(\Ff\)-reflection is what makes the two raising slots mirror images; the Doppler reflection is its image under the bilinear map, and forces every diagonal element (\(\ell=\ell''\)) to be a polynomial in the diffusion operator \(\Dv = \Oh^2-3\Oh\).

Generation from the seed

Fill the \(\Ff\)-table from \(\Ff_{00}\): raise the row to build column \(b=0\), then raise the column; assemble any \(\Do\) by the bilinear form. Verified end-to-end — from the seed alone, with no kernels, integrals, or boosts — for e.g. \(\Do_{313}, \Do_{212}, \Do_{303}, \Do_{222}\) V4, exact:

F(0,0,q) = J[2+q]/(2p)                                   # the seed
F(a,0,q) = (1/(C_a p))(γ F(a-1,0,q) - F(a-1,0,q+1)) - (C_{a-1}/C_a) F(a-2,0,q)   # row-raise
F(a,b,q) = -(1/(C_b p))(γ F(a,b-1,q) - F(a,b-1,q-1)) - (C_{b-1}/C_b) F(a,b-2,q)  # col-raise
D(l,l',l'',q) = ((-1)^(l'+l'') / γ) F(l,l',q) F(l',l'',q-1)                      # assemble

Termination: each raise lowers an index; the seed carries \(1/(2p)\), so \(\Do_{\ell\ell'\ell''}\) lands in \(\frac{1}{4\gamma p^{\,2+\ell+2\ell'+\ell''}}\,\mathbb Q[\gamma,p]\)-span\(\{J_{a+q}J_{b+q}\}\) — the N8 structure theorem, now the output of a two-line algorithm.

Two levels of use

  • Bracket level (simplest). Work in \(\Ff\): one seed, two recurrences, one reflection. Assemble \(\Do\) at the end. This is the whole calculus.
  • Pure-physical level. If every intermediate must be a physical \(\Do\), use the bilinear identity + gauge to trade brackets for \(\Do\)'s at shifted \(q\): the outer axes close linearly (paper P0/P1), the middle index only nonlinearly (paper P3), and the rank-one structure (P2) fills the rest.
